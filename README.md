# Pinterest Clone

This is a Pinterest clone built using Express Generator.

## Description

This project aims to replicate the core functionalities of Pinterest, allowing users to discover, save, and share images.

## Features

- User Authentication: Users can sign up, log in, and log out.
- Image Upload: Users can upload images.
- Image Saving: Users can save images to their profile.
- Image Sharing: Users can share images with others.
- Image Discovery: Users can discover new images through browsing.
- Responsive Design: The application is designed to be responsive across different devices.

## Technologies Used

- Express.js: A web application framework for Node.js.
- Express Generator: A tool to quickly scaffold an Express application.
- MongoDB: A NoSQL database for storing application data.
- Mongoose: An Object Data Modeling (ODM) library for MongoDB and Node.js.
- Passport.js: An authentication middleware for Node.js.
- Multer: A middleware for handling multipart/form-data, used for uploading images.
- EJS: Embedded JavaScript templates for generating HTML markup.
- etc.

## Installation

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Set up environment variables (e.g., MongoDB URI, session secret).
4. Start the server using `npm start`.

## Usage

1. Visit the application in your web browser.
2. Sign up for an account or log in if you already have one.
3. Explore images, upload your own, save favorites, and share with others.
